当你调用构造函数时，你实际上并没有创建与语言代码相对应的 Lang 对象。事实上，Lang只是一个存储下标的类。这样可以大大提高内存利用效率 调用构造函数时，会首先检查Lang的伴生对象的baseLangs中是否存在与该语言代码对应的BaseLang。如果是，则将找到的“BaseLang”对象的索引分配给“Lang”中的“index”。如果没有，请创建一个新的并添加它。同样，将Lang中的index分配给新创建的'BaseLang'对象
